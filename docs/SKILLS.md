# Skill Matrix

This starter vendors a curated set of repo-local fallback skills for the first ready profiles.

## Why Curate Instead Of Copying Everything

- keeps the starter smaller and easier to understand
- reduces maintenance drift
- gives beginners a shorter, more useful toolbelt
- still leaves room for global skills and later expansion

## Vendored Skills In This Starter

- `brainstorming`
- `gh-fix-ci`
- `gh-address-comments`
- `writing-plans`
- `verification-before-completion`
- `systematic-debugging`
- `test-driven-development`
- `requesting-code-review`
- `playwright`
- `skill-creator`

## Skill Creator

`skill-creator` is vendored so a user can extend the generated repo with their own project-specific skills when needed, without depending only on global machine state.
